# C++ Finance Project

Academic group project for the course **C++ for Finance** (2025).

## Project description
This project implements option pricing models in C++, including:
- Black-Scholes model
- Cox-Ross-Rubinstein (CRR) binomial model
- Monte Carlo pricing
- European, Digital, Asian and American options

The goal is to design clean, reusable C++ classes following object-oriented principles.
